exp-ang-sio
===========

[Express 4](https://github.com/visionmedia/express) + [Angular](https://angularjs.org/) + [Socket.io](http://socket.io/) minimal example application



## Installation

    $ git clone https://github.com/pwldp/exp-ang-sio


    $ cd exp-ang-sio


    $ npm install .

    

## Run

    $ node app.js
